## 👨‍💻 Sobre mí

Soy desarrollador de software junior y estudiante de tercer año de Bachillerato Técnico Vocacional en Desarrollo de Software en el Colegio Español Padre Arrupe.

Me enfoco en la creación de aplicaciones web y móviles modernas, interfaces intuitivas, sistemas de gestión de datos e integración de herramientas de inteligencia artificial.

He trabajado en proyectos académicos, personales y freelance utilizando tecnologías como **React, Next.js, React Native, TypeScript, Firebase, PHP, MySQL y C#**.
